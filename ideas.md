# 개발자 포트폴리오 웹사이트 - 디자인 아이디어

## 선택된 디자인 접근법: Modern Playful Minimalism

### 디자인 철학
UI 디자인 이미지에서 영감을 받아, **현대적이면서도 장난스러운 미니멀리즘** 스타일을 채택합니다. 이는 전문성과 창의성의 균형을 맞추며, 깔끔한 레이아웃 속에서 대담한 타이포그래피와 유쾌한 시각 요소들을 조화시킵니다.

### 핵심 원칙

1. **대담한 타이포그래피**: 큰 헤드라인(Hero 섹션의 "I build digital experiences")과 일반 텍스트의 명확한 계층 구조
2. **전략적 공백**: 섹션 간 충분한 여백으로 호흡감 있는 레이아웃 구성
3. **색상 강조**: 밝은 파란색(Primary Blue #2563EB)을 주요 액센트로 사용하여 CTA와 중요 요소 강조
4. **유기적 요소**: 손그림 같은 데코레이션(밑줄, 별, 박스 등)으로 따뜻하고 인간적인 느낌 전달

### 색상 철학

| 역할 | 색상 | 용도 |
|------|------|------|
| 배경 | White (#FFFFFF) | 깔끔하고 전문적인 기본 배경 |
| 주요 텍스트 | Dark Slate (#1F2937) | 본문 및 제목 |
| 강조색 | Bright Blue (#2563EB) | 버튼, 링크, 주요 강조 |
| 보조색 | Light Gray (#F3F4F6) | 섹션 배경, 카드 배경 |
| 악센트 | Warm Orange (#F59E0B) | 데코레이션, 보조 강조 |

### 레이아웃 패러다임

- **비대칭 구성**: Hero 섹션에서 텍스트는 좌측, 이미지/데코는 우측에 배치
- **카드 기반 프로젝트**: 프로젝트 섹션은 3열 그리드로 표현
- **섹션 분리**: 각 섹션을 명확하게 구분하되, 흐름은 자연스럽게 유지

### 시그니처 요소

1. **손그림 데코**: 밑줄(underline), 별(*), 박스, 화살표 등으로 장난스러운 분위기 연출
2. **컬러풀한 이미지 콜라주**: 포트폴리오 이미지들을 겹쳐서 배치하여 시각적 흥미 유발
3. **버튼 스타일**: 파란색 배경의 주요 버튼과 아웃라인 스타일의 보조 버튼

### 상호작용 철학

- **호버 효과**: 버튼과 프로젝트 카드에 부드러운 스케일 및 그림자 변화
- **스크롤 애니메이션**: 섹션이 뷰포트에 진입할 때 페이드인 효과
- **링크 강조**: 텍스트 링크에 밑줄 애니메이션

### 애니메이션 가이드

- **진입 애니메이션**: 섹션은 아래에서 위로 부드럽게 페이드인 (duration: 0.6s)
- **호버 상태**: 프로젝트 카드는 스케일 1.05로 확대, 그림자 강화 (duration: 0.3s)
- **버튼 상호작용**: 클릭 시 약간의 스케일 다운 후 복구 (duration: 0.2s)

### 타이포그래피 시스템

| 요소 | 폰트 | 크기 | 가중치 |
|------|------|------|--------|
| 메인 헤드라인 | Poppins | 64px | 700 (Bold) |
| 섹션 제목 | Poppins | 48px | 700 (Bold) |
| 소제목 | Poppins | 24px | 600 (Semi-Bold) |
| 본문 | Inter | 16px | 400 (Regular) |
| 작은 텍스트 | Inter | 14px | 400 (Regular) |

**폰트 선택 이유**: Poppins는 현대적이고 친근한 느낌을 주며, Inter는 가독성이 뛰어나 본문에 적합합니다.
