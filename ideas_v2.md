# 개발자 포트폴리오 웹사이트 - 디자인 아이디어 v2

## 선택된 디자인 접근법: Handcrafted Collage

### 디자인 철학
UI 디자인 이미지를 정확히 기준으로, **손으로 만든 콜라주** 스타일을 채택합니다. 
창의적, 젊은, 예술적, 실험적, 촉각적 분위기로 전문성과 창의성을 동시에 표현합니다.

### 핵심 원칙

1. **다양한 타이포그래피**: 여러 폰트 조합으로 레이어드 창의성 표현
2. **자유로운 배치**: 엄격한 그리드 대신 자연스러운 산란 배치
3. **데코레이션 요소**: 오렌지, 레몬, 음식, 종이, 테이프, 스티커 등 실제 물체 이미지
4. **비대칭 구성**: 겹침, 회전, 불완전한 정렬로 동적 느낌
5. **촉각적 디테일**: 종이 질감, 테이프 효과, 찢어진 가장자리, 손그림

### 색상 철학

| 용도 | 색상 | 코드 |
|------|------|------|
| 배경 | White | #FFFFFF |
| 주요 텍스트 | Dark Gray | #1F2937 |
| 주요 강조 | Bright Blue | #2563EB |
| 보조 강조 | Warm Orange | #F59E0B |
| 액센트 | Warm Yellow | #FBBF24 |
| 중성 | Light Gray | #F3F4F6 |

### 타이포그래피 시스템

| 요소 | 폰트 | 용도 |
|------|------|------|
| 메인 헤드라인 | Fredoka Bold | 큰 제목, 임팩트 있는 텍스트 |
| 섹션 제목 | Caveat (손글씨) | 섹션 제목, 장난스러운 느낌 |
| 서브 헤드 | Poppins Bold | 중간 크기 제목 |
| 본문 | Inter | 일반 텍스트, 설명 |
| 강조 | Playfair Display | 특수 강조, 인용문 |

### 레이아웃 패러다임

- **비대칭 구성**: 요소들이 자유롭게 배치됨
- **겹침과 회전**: 사진 카드처럼 회전되고 겹쳐진 요소들
- **경계 무시**: 데코레이션이 컨테이너를 넘어감
- **자연스러운 산란**: 마치 손으로 배치한 것처럼 보이는 배치
- **섹션 구분**: 찢어진 종이, 테이프 효과로 섹션 분리

### 시그니처 요소

1. **데코레이션 이미지**: 오렌지, 레몬, 음식, 종이, 조개, 테이프, 스티커
2. **사진 카드**: 프로젝트 이미지를 마치 폴라로이드 사진처럼 표현
3. **손그림 요소**: 밑줄, 화살표, 박스, 별 등 손으로 그린 느낌
4. **종이 질감**: 배경과 카드에 종이 질감 추가
5. **테이프 효과**: 마스킹 테이프로 요소를 고정한 느낌

### 상호작용 철학

- **호버 효과**: 카드가 살짝 들리는 느낌 (그림자 강화)
- **클릭 피드백**: 약간의 회전 변화
- **스크롤 애니메이션**: 요소들이 자연스럽게 나타남
- **장난스러운 움직임**: 약간의 흔들림 애니메이션

### 애니메이션 가이드

- **진입 애니메이션**: 요소들이 회전하며 페이드인 (duration: 0.8s)
- **호버 상태**: 카드가 살짝 들리고 그림자 강화 (duration: 0.3s)
- **스크롤 트리거**: 요소가 뷰포트에 진입할 때 애니메이션
- **미묘한 흔들림**: 일부 데코레이션 요소에 부드러운 흔들림 효과

### 폰트 선택

```
Google Fonts:
- Fredoka: Bold (700) - 메인 헤드라인
- Caveat: Regular (400) - 손글씨 섹션 제목
- Poppins: Bold (700) - 서브 헤드라인
- Inter: Regular (400) - 본문
- Playfair Display: Regular (400) - 강조 텍스트
```

### 데코레이션 요소 배치

#### Hero 섹션
- 우측 상단: 레몬 슬라이스 (회전 5도)
- 우측 하단: 오렌지 반쪽 (회전 -8도)
- 좌측 하단: 음식 이미지 (회전 3도)
- 배경: 종이 질감

#### About 섹션
- 좌측: 피자 슬라이스 (회전 -12도)
- 우측 상단: 조개 (회전 8도)
- 배경: 가벼운 그리드 패턴

#### Projects 섹션
- 프로젝트 카드들이 겹쳐서 배치
- 각 카드: 마스킹 테이프 효과
- 카드 뒤에 종이 질감
- 일부 카드는 회전 (rotate(-6deg), rotate(4deg) 등)

#### Contact 섹션
- 배경: 파란색 + 종이 질감
- 데코: 테이프, 스티커 느낌의 요소들
